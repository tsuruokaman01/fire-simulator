# 🔥 FIREシミュレーター

昇進・教育費・習い事まで設定できる、日本一解像度の高い無料FIREシミュレーター。

## 特徴

- **⚡ かんたんモード** — 6項目入力でFIREプランを即把握
- **⚙️ 詳細モード** — 昇進マイルストーン・退職金・教育費・保険・積立ペースを年齢別に細かく設定
- **🧮 年収→手取り早見表** — 日本の税制に基づいた正確な計算
- **📊 リアルタイムグラフ** — 純資産推移・収支・支出内訳を可視化
- **📱 SNSシェア** — X(Twitter)・LINEへ結果をシェア

## 使い方

### ローカルで起動する場合

```bash
pip install -r requirements.txt
streamlit run app.py
```

ブラウザで `http://localhost:8501` が開きます。
